StockPredict API

Welcome to the StockPredict API. This API provides stock data, technical indicators, and price predictions for Indian stocks.

Available Endpoints:

GET /api/stock-symbols/

Get a list of available stock symbols.

Optional query parameters:

  • exchange: Filter by exchange (e.g., NSE, BSE)

GET /api/stock-data/{symbol}/

Get historical data for a specific stock.

Optional query parameters:

  • timeframe: Time period (e.g., 1d, 1w, 1m, 3m, 6m, 1y, 5y)

POST /api/technical-indicators/

Calculate technical indicators for a stock.

Request body:

{
  "symbol": "RELIANCE",
  "timeframe": "1y",
  "indicators": ["sma", "ema", "rsi", "macd", "bollinger_bands"]
}

GET /api/prediction-models/

Get a list of available prediction models.

POST /api/predict/

Make price predictions for a stock.

Request body:

{
  "symbol": "RELIANCE",
  "model_type": "linear",
  "days_to_predict": 30
}

GET /api/market-overview/

Get an overview of Indian market indices.

GET /api/search-stocks/

Search for stocks by name or symbol.

Query parameters:

  • q: Search query (e.g., "Reliance", "HDFC")
